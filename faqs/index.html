PrintPGN

This site was created to make analysing chess games over the board easier by formatting the moves in a more readable way. Once you have a PGN you can use PrintPGN to create a table formatted ready for print.

How to use PrintPGN

Paste your PGN into the text area. Once the PGN is successfully parsed the chess games and print options will be shown. Use the options to configure what is printed and the layout of the moves table.

The title, meta data, and comments for each game are editable. However, all edits will be lost if you adjust any of the print options before printing.

Note: You may need to use the setting on your printer to fine tune the scale the games are printed at.
